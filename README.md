# Aura Control

Aura-Control es un ecosistema de software diseñado para simular el control de alta precisión de actuadores en telescopios de gran escala (como el ELT) o sistemas de diagnóstico en aceleradores de partículas. El objetivo principal es demostrar la capacidad de integrar sistemas de alto rendimiento en C++ con lógica de negocio/IA flexible en Python, exponiendo los resultados a través de interfaces modernas y seguras.
